Frequently Asked Questions

‚ùì How is this better than a spreadsheet?

Automatic FIFO calculations, no broken formulas, proper partial box handling, and organized location hierarchy. Access from any browser with no installation needed. Plus, no one can accidentally overwrite your data or break your formulas. It's purpose-built for ammunition, not adapted from Excel.

üîí Is my data private and secure?

Yes. Industry-standard BCrypt password hashing, HTTPS connections in production, field-level encryption for personal data, and row-level data isolation between users (every query filters by your user ID). Your inventory data is never shared or sold.

üì± Is there a mobile app?

The web application is accessible from mobile browsers - no separate app needed. Mobile optimization is actively being improved based on beta user feedback. Native iOS/Android apps are on the roadmap.
